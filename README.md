# Breast-Cancer-KNN-Classification
Machine Learning project that predicts whether a breast tumor is benign or malignant using the K-Nearest Neighbors (KNN) algorithm.


Breast Cancer Classification using K-Nearest Neighbors (KNN)

Overview

This project uses the Breast Cancer Wisconsin Dataset to classify tumors as:

* Benign (Class = 2)
* Malignant (Class = 4)

using the K-Nearest Neighbors (KNN) machine learning algorithm.

Problem Statement

The goal is to build a classification model that can assist in identifying whether a breast tumor is cancerous or non-cancerous based on cell characteristics extracted from a fine needle aspirate.

Dataset Features

* Clump Thickness
* Cell Size Uniformity
* Cell Shape Uniformity
* Marginal Adhesion
* Single Epithelial Cell Size
* Bare Nuclei
* Bland Chromatin
* Normal Nucleoli
* Mitoses

Target:

* 2 = Benign
* 4 = Malignant

Project Workflow

1. Data Loading
2. Data Inspection
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. K Value Optimization
6. KNN Model Training
7. Model Evaluation
8. Test Predictions
